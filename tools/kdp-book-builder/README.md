# KDP Book Builder

A specialized utility designed to streamline the process of preparing manuscripts for Kindle Direct Publishing (KDP).

## 🚀 Purpose
This tool allows authors to input their manuscript text and format it into a professional layout suitable for KDP submission. It handles basic formatting and provides a clean interface for content management.

## 🛠️ Tech Stack
- **HTML5/CSS3**: Core structure and styling.
- **JavaScript**: Logic for text processing and formatting.
- **jsPDF**: Client-side PDF generation.
