# Markdown to PDF Converter

A fast and efficient web tool to convert Markdown documents into clean, professional PDF files.

## 🚀 Features
- **Real-time Preview**: See how your document looks as you type.
- **Marked.js**: High-performance Markdown parsing.
- **jsPDF & html2canvas**: Reliable client-side PDF generation.
- **Minimalist UI**: Focused on productivity and ease of use.

## 📁 Usage
1. Paste or type your Markdown content into the editor.
2. Review the live preview.
3. Click "Generate PDF" to download your document.
