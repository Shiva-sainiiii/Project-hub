# Project Hub

A professional collection of web tools, games, and portfolio templates organized into a single, clean repository.

## 🚀 Overview

This hub serves as a central repository for various web-based projects. It is structured to be easily navigable, with clear separations between different types of applications.

## 📂 Structure

- **[games/](./games/)**: Interactive games built with vanilla JavaScript and HTML5 Canvas.
- **[tools/](./tools/)**: Practical utility tools for developers and creators.
- **[portfolios/](./portfolios/)**: A collection of high-quality portfolio templates and personal sites.
- **[assets/](./assets/)**: Shared global assets like favicons and brand logos.

## 🛠️ Projects Included

### Games
- **[Snake Rush](./games/snake-game/)**: A modern, high-performance take on the classic Snake game with power-ups and AI opponents.

### Tools
- **[KDP Book Builder](./tools/kdp-book-builder/)**: A tool to help format manuscripts for Kindle Direct Publishing.
- **[Markdown to PDF](./tools/markdown-to-pdf/)**: Convert Markdown text into professional PDF documents.
- **[Creative Portfolios](./tools/creative-portfolios/)**: Four unique, high-end 3D portfolio experiences (Claude, Copilot, Gemini, Meta).

### Portfolios
- **[Portfolio V1](./portfolios/portfolio-v1/)**: A clean and modern personal portfolio template.

## 📖 How to Use

Simply open the `index.html` file in your browser to access the main hub landing page, from which you can navigate to any of the included projects.
