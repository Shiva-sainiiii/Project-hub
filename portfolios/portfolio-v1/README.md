# Portfolio V1

A modern, responsive personal portfolio template designed for developers and designers.

## ✨ Highlights
- **Clean Aesthetic**: Focused on typography and whitespace.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Interactive Elements**: Smooth transitions and hover effects.
- **Customizable**: Easy to update with your own projects and experience.

## 📁 Files
- `index.html`: Main structure.
- `styles.css`: Custom styling.
- `script.js`: Interactive components.
- `favicon.png`: Brand asset.
