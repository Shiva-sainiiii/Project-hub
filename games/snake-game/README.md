# Snake Rush

A modern, high-performance take on the classic Snake game, built with Vanilla JavaScript and HTML5 Canvas.

## ✨ Features
- **Power-ups**: Collect magnets, attack boosts, and lifelines.
- **AI Opponents**: Compete against 10 smart AI snakes.
- **Customization**: Choose from multiple snake designs (Multicolour, Fatty, Thin, Designer).
- **Immersive Audio**: Proximity-based sound effects and background music.
- **Responsive Controls**: Optimized for both mouse and touch input.

## 📁 Structure
- `index.html`: The game's entry point.
- `style.css`: Visual styling and layout.
- `game.js`: Core game logic, physics, and audio management.
- `assets/`: Sound effects and music files.
